# vue-multi

It is a vue template for multipage project powered by [vue-cli](https://github.com/vuejs/vue-cli).

## Usage

``` bash
$ npm install -g vue-cli
$ vue init PaicFE/vue-multi my-project
$ cd my-project
$ npm install
$ npm run build:dll
$ npm run dev
$ npm run build
```

## Features

- Vue Multipage 
- DLL JS Package
- ejs for html
